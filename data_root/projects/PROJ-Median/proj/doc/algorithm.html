The median selection algorithm finds the median of a given array

Input: An array of integers.
Output: Median of the input array.
Test of correctness:     The execution of an algorithm for a given array is correct if the returned median is the (array.size/2)-smallest element.
Implementation: To implement the median selection algorithm the author has to write a java class extending the MedianAbsAlgorithm class
