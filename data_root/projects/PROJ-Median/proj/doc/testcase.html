A test case used in the median selection problem is a list of any comparable data. Usually the elements of the list are of the same type (integers, real numbers, strings, ...).

In this project a test case is a one-dimensional fixed-sized array of integers. The aim of the algorithm is to find the median of this array. For the Java programs the test case is declared with

int[] arrayToSelectFrom = new int[n];
where n is the size of the input array.
